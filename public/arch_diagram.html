Component Architecture

01
🧭
StudioNavbar
<StudioNavbar />

Top bar ثابت فوق الـ layout. يعمل hide-on-scroll عبر Zustand flag `isNavbarHidden`. Glassmorphism blur مع إخفاء سلس.

position: fixed z-index: 50 backdrop-blur
02
🗂
Left Navigation Rail
<SessionSidebar />

شريط يسار ضيّق `56px`. فيه icon buttons للـ navigation بين collections/sessions. يفتح/يغلق بـ state واحدة.

width: 56px flex-col Zustand
03
📜
Virtuoso Scroller
react-virtuoso

Virtual scroll engine — ما يرندرش غير الـ items الظاهرة في الـ viewport. يدعم dynamic heights و infinite loading بأداء عالي.

windowed rendering overflow-y: auto rAF throttle
04
🖼
Media Card
<MediaGridItem />

كل card فيها: thumbnail، عنوان collection، drag handle، وlinks للـ collection page. تدعم DnD عبر `aria-roledescription="draggable"`.

468×400px draggable aspect-ratio
05
✦
Floating PromptBar
<PromptBar />

Fixed bottom bar `absolute bottom-10`. يطفو فوق المحتوى مع pointer-events-none على الـ wrapper ليمرّر الـ clicks للـ grid خلفه.

z-index: 30 glassmorphism max-w: 650px
06
⚡
Drag Overlay
<DnD State>

عند drag من الـ gallery، layer شبه-شفاف يغطي الـ main area مع `backdrop-blur`. يُفعَّل عبر `isDraggingGalleryItem` في Prompt Store.

z-index: 25 framer-motion bg-black/60
